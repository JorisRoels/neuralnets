Neuralnets documentation
========================
Welcome to the documentation page of the neuralnets library! This library will help you accelerate the training process of your image-based deep learning tasks. The neuralnets library is built on top of PyTorch and offers GPU accelerated data augmentation, implementation of various networks, flexible I/O, etc.
